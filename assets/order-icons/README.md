# Order Icons

Ikon emoji untuk template order Reiizam Store.

Mapping:
- `rocket.svg` -> rocket / form header
- `category-folder.svg` -> folder / kategori
- `product-package.svg` -> package / produk
- `duration-hourglass.svg` -> hourglass / durasi
- `price-moneybag.svg` -> money bag / harga
- `code-key.svg` -> key / kode
- `thanks-pray.svg` -> thanks / penutup
